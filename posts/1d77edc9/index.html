VSCode 常用快捷键

发表于 2022-09-15 更新于 2022-09-24 分类于 VSCode 阅读次数：
VSCode常用快捷键

代码格式化

  • shift + alt +f

快速复制一行

  • Shift + Alt + ↓/↑

  • 在某行直接Ctrl + C + Ctrl + V

快速定位到某一行

  • Ctrl + G

选中单词

  • 在单词首或尾 Ctrl + Shift + ←/→

选定多个相同的单词

  • 选定某词直接 Ctrl + D

全局替换某个单词

  • Ctrl + H

选择某个区块

  • 按住 Alt + Shift 选择或拖动鼠标

放大缩小整个编辑器界面

  • Ctrl + +/-

光标定位到单词首/单词尾

  • Ctrl + ←/→

光标定位到行首/行未

  • Home / End

添加多个光标

  • Ctrl + Alt + ↑/↓
